# Logger

Реализация динамической библиотеки для записи сообщений в журнал с разными уровнями важности и консольного многопоточного приложения для демонстрации работы библиотеки.

## Реализация

- Динамическая библиотека `liblogger.so`.
- Поддержка уровней важности сообщений:
  - `DEBUG`
  - `INFO`
  - `ERROR`
- Фильтрация сообщений по уровню важности.
- Запись в текстовый файл:
  - времени получения сообщения;
  - уровня важности;
  - текста сообщения.
- Возможность изменения уровня важности во время работы приложения.
- Потокобезопасная передача сообщений между пользовательским потоком и потоком записи.
- Использование стандартной библиотеки C++17.

## Сборка

Сборка библиотеки и приложения:

```bash
make
```

Очистка временных файлов:
```bash
make clean
```

## Запуск

```
./main <log_file> <default_level>
```

Пример:
```
./main application.log INFO
```

## Работа приложения

Сообщение без указания уровня

```
message
```

Сообщение с уровнем
```
INFO message
```

Изменение уровня сообщений
```
/set LEVEL
```

Завершение приложения
```
/exit
```

Пример файла журнала
```
2026-07-20 17:30:12 [INFO] Сервер запущен
2026-07-20 17:30:20 [ERROR] Доступ к базе данных закрыт
```
